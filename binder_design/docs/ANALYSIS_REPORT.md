# NKG7 extracellular-loop mini-binder campaign — analysis report

**Goal.** Build a reusable, well-documented de novo mini-binder design pipeline
and use it to design binders against the **extracellular loops** of NKG7 (never
the transmembrane or cytoplasmic regions), in both **human and mouse**, then
prove the pipeline end-to-end on the marvin HPC cluster with a smoke test.

**Status.** Pipeline built, installed, and verified on marvin; smoke test run
end-to-end across all 6 campaign arms. This report documents the biology, the
design decisions, the pipeline, and the smoke-test results, and specifies how to
launch the full production campaign.

Companion documents: [`DESIGN_DECISIONS.md`](DESIGN_DECISIONS.md) (rationale for
every methodological choice) and [`README.md`](README.md) (how to run the
pipeline on a new target).

---

## 1. Target biology

NKG7 (UniProt **Q16617** human, **Q99PA5** mouse; 165 aa) is a 4-pass
transmembrane protein. Transmembrane helices span residues 9–29, 61–81, 92–112,
133–153; both termini are cytoplasmic. Only two short segments are
extracellular and therefore accessible to a binder on an intact cell:

| Loop | Human residues | Length | Human sequence | Mean pLDDT |
|------|----------------|--------|----------------|-----------|
| ECL1 | 30–60  | 31 aa | EAVGPTHSAHSGLWPTGHGDIISGYIHVTQT | 85 |
| ECL2 | 113–132 | 20 aa | TSERWDQPPHPQIQTFFSWS | 87 |

The two loops are spatially adjacent (centroid–centroid 11.3 Å; closest CA
approach 4.6 Å), both projecting ~+23 Å above the membrane plane, so they can be
targeted individually **or** as a single composite epitope.

**Key structural fact driving the method:** neither loop contains a disulfide —
all five cysteines (4, 19, 80, 145, 155 human) are in TM/cytoplasmic regions —
so the loops are conformationally mobile. This motivates ensemble targeting
(§3 below).

**Cross-species conservation:** ECL1 61 % identical (19/31), ECL2 70 % (14/20)
human↔mouse; conserved cores `SAHSGLWPT`/`GYIHVTQ` (ECL1) and `FFSWS` (ECL2).
Moderate conservation means some binders may be cross-reactive, but this must be
tested, not assumed — hence separate human and mouse campaigns.

## 2. Design decisions (summary)

Full rationale, rejected alternatives, and residual risks are in
`DESIGN_DECISIONS.md`. The eight decisions:

1. **Target the extracellular loops only** — TM/cytoplasmic epitopes are
   inaccessible on live cells; existing NKG7 antibodies bind the intracellular
   C-terminus and cannot label intact cells.
2. **Keep the full-length target, restrict the interface to loop hotspots** —
   the TM bundle acts as an implicit steric wall so binders approach only from
   the extracellular side; excising loops into peptides would lose that context.
3. **Design against a loop-conformer ensemble, not one AF snapshot** — the loops
   are mobile (no disulfides), so binders are profiled across sampled
   conformations rather than over-fit to one uncertain snapshot.
4. **Three epitopes per species (ECL1, ECL2, composite)** — maximise hit
   chances; composite is the highest-probability arm (largest interface), ECL2
   alone the lowest (20 aa) but kept as an inexpensive hedge.
5. **Separate human and mouse campaigns, cross-reactivity tested post hoc** —
   moderate loop conservation makes cross-reactivity plausible but not
   guaranteed.
6. **De novo mini-binders (RFdiffusion → ProteinMPNN → Boltz-2)**, ~50–65 aa —
   the modality the user selected; nanobody grafting deferred.
7. **In silico validation only at this stage** — no experimental NKG7 structure
   exists; all conclusions are model-based and must be confirmed at the bench.
8. **Validator = Boltz-2 (pip-installed)** — forced by a marvin hardware
   constraint (§5): the EasyBuild AlphaFold/ColabFold/Boltz-1 modules crash
   (SIGILL) on the AMD GPU nodes.

## 3. Loop-conformer ensemble & flexibility profile

Loop ensembles were generated by restrained molecular dynamics (OpenMM, implicit
solvent GBn2; TM+cytoplasmic core position-restrained, loops free at 310 K),
12 conformers per species, on a marvin A40 GPU. Core-aligned RMSF confirms the
intended behaviour:

| Region | Human RMSF (mean / max) | Mouse RMSF (mean / max) |
|--------|--------------------------|--------------------------|
| Core (restrained) | 0.12 Å | 0.12 Å |
| ECL1 | 0.81 / 2.01 Å | 0.87 / 2.40 Å |
| ECL2 | 0.63 / 1.58 Å | 0.64 / 1.58 Å |

The core is rigid; both loops are mobile, ECL1 more than ECL2 in both species —
consistent with ECL1 being longer and less structurally constrained. This is the
conformational space the design campaign profiles against. The overlay figure
([`figures/loop_ensemble_overlay.png`](figures/loop_ensemble_overlay.png)) shows
the rigid TM bundle in grey with ECL1 (red) and ECL2 (blue) conformers fanned
out.

## 4. Pipeline

Config-driven; a single YAML (`configs/nkg7_campaign.yaml`) defines the target,
the 6 arms, and all parameters. Three stages, each verified independently on
marvin:

1. **RFdiffusion** — de novo backbone generation against the fixed full-length
   target, loop residues as hotspots (`Complex_base_ckpt.pt`; contig
   `[A1-165/0 50-65]` = full target + 50–65-residue binder).
2. **ProteinMPNN** — sequence design on the binder chain, target chain fixed,
   soluble model (`v_48_020`).
3. **Boltz-2** — refold each binder:target complex, score interface (ipTM),
   fold confidence (pLDDT), binder ptm, and design-vs-refold CA-RMSD.

Filtering ranks by `composite_score = 0.45·ipTM + 0.30·pLDDT + 0.15·binder_ptm +
0.10·rmsd_term`; pass thresholds ipTM ≥ 0.5, pLDDT ≥ 0.7, binder_ptm ≥ 0.5,
RMSD ≤ 2 Å. Everything is orchestrated as a SLURM DAG (RFdiffusion → afterok →
ProteinMPNN → afterok → Boltz), arms in parallel.

## 5. Hardware constraint discovered (and worked around)

marvin's `mlgpu` (A40) nodes have **AMD CPUs with AVX2 but no AVX-512**, while
the login node is Intel with AVX-512 and the site's `easybuild-INTEL` module
stack is AVX-512-compiled. The AlphaFold/ColabFold and Boltz-1 **modules crash
immediately with SIGILL (illegal instruction) on the GPU nodes.** This was
diagnosed by a CPU-flag probe on a compute node after two independent toolchains
failed identically. Resolution: every ML tool is **pip-installed** (generic
AVX2 wheels) in a dedicated conda env rather than `module load`ed. This is the
single most important operational fact for anyone reusing this pipeline on
marvin, and it is documented in the README and DESIGN_DECISIONS §8.

## 6. Smoke-test results

Scale: 50 RFdiffusion backbones/arm at diffusion T=25, 8 ProteinMPNN sequences,
Boltz capped to 8 complexes/arm (the 8 sequences of the first backbone). This
scale **validates pipeline mechanics, not final binders** — T=25 backbones are
deliberately low-diversity and low-quality.

All 6 arms ran end-to-end: **48 designs scored, 0 pass all production
thresholds** — expected at this scale. pLDDT is the limiting metric (all arms
~0.55–0.62, below the 0.7 gate), which is the signature of low-diversity smoke
backbones, not a pipeline fault. Per-arm best ipTM:

| Arm | best ipTM | best pLDDT | best composite | pass |
|-----|-----------|------------|----------------|------|
| mouse ECL2 | 0.615 | 0.614 | 0.674 | 0 |
| human ECL2 | 0.625 | 0.618 | 0.648 | 0 |
| human composite | 0.676 | 0.600 | 0.588 | 0 |
| mouse composite | 0.478 | 0.615 | 0.574 | 0 |
| human ECL1 | 0.358 | 0.616 | 0.521 | 0 |
| mouse ECL1 | 0.256 | 0.624 | 0.486 | 0 |

Metric distributions per arm are in
[`figures/smoketest_metrics.png`](figures/smoketest_metrics.png); the full
ranked table is [`smoketest_candidates_ranked.csv`](smoketest_candidates_ranked.csv).
At smoke scale the arm ranking is dominated by sampling noise, not the a priori
epitope feasibility — that ordering will only be meaningful at production scale.

**What the smoke test proves:** RFdiffusion, ProteinMPNN, and Boltz-2 all run on
marvin's A40 nodes; the SLURM DAG chains correctly; outputs are harvested and
ranked with all interface metrics (including design-vs-refold RMSD) populated
for every design. The pipeline is ready for a production campaign.

## 7. Path to production

1. Switch to `--mode production` (2000 backbones/arm, T=50, 16 sequences, no
   Boltz cap → all complexes folded), on `mlgpu_short`/`mlgpu_medium`.
2. Expect a small fraction to pass the interface filter; the composite arms are
   the highest-probability, ECL2-alone the lowest.
3. For passing designs: (a) re-fold each species' top binders against the other
   species' target for a **cross-reactivity** read (ipTM > 0.5 against both →
   predicted cross-reactive); (b) run a second orthogonal validator (e.g.
   Chai-1) for consensus; (c) inspect the interface for the intended loop
   contacts.
4. All in silico — synthesise and test the top candidates experimentally before
   any conclusion.

## 8. Limitations

- No experimental NKG7 structure exists; every step builds on an AlphaFold
  model, and the ensemble is force-field-sampled from that model.
- Smoke-scale results carry no biological signal; production scale is required
  for real candidates.
- Boltz-2 confidence (ipTM/pLDDT) is a predictor of complex plausibility, not a
  measurement of affinity; experimental validation is mandatory.
- The TM bundle in the model approximates but does not reproduce the true
  lipid-embedded environment.
